# plugin-js-tutorial

initialize a virtual enviroment

```
python3 -m venv demo_venv
```

activate virtual-enviroment:

```
source demo_venv/bin/activate
```

update pip

```
pip install --upgrade pip
```

istall wheel (otherwhise some packges may give you an error)

```
pip install wheel
```
